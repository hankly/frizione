<static> {Array} frizione.service.checkJsDocParams(params)
Checks the given JsDoc parameters. Valid parameters:
  • src the source file path (can be used multiple times).
  • a include all functions, even undocumented ones.
  • d specifies the output directory.
  • e specifies the file encoding to use.
  • n ignore all code, only document comments with @name tags.
  • p include symbols tagged as private, underscored and inner symbols.
  • s suppress source code output.
Parameters:
{Object} params
the parameters to check.
Returns:
{Array} an array of errors, or null if no errors occurred.
<static> {Array} frizione.service.checkMinifyParams(params, file, to, gzip)
Checks the given minify parameters. Valid parameters:
  • to the file path to write the minified file to.
  • cs the character set of the input and output file (default is UTF-8).
  • gzip the file path to write the gzipped file to.
  • mc the maximum number of characters before a forced line break (default is 0).
  • munge reduce function and variable names (default is false).
  • ps preserve semicolons (default is true).
  • opt use micro optimisations (default is true).
  • v display errors and warnings (default is true).
Parameters:
{Object} params
the parameters to check.
{String} file
the original file path.
{String} to
the amalgamated output file path.
{String} gzip
the amalgamated gzip file path.
Returns:
{Array} an array of errors, or null if no errors occurred.
<static> {Array} frizione.service.checkTestParams(params, file, to, gzip)
Checks the given test parameters. Valid parameters:

  • json the file path to write the unit test JSON file to.
