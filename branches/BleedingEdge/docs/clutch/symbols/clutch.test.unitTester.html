Method Detail
{Object} check()
Checks the status of the unit tests being run. Returns a status object with four fields:
  • complete - a boolean which is set to true when all tests are complete.
  • abend - a string which is set if an abnormal condition (such as unit testing timeout) occurred.
  • index - a number indiciating the index of the current unit test being run.
  • total - a number indicating the total number of unit tests to be run.
Returns:
{Object} the current unit test status.
run()
Runs the unit tests.
{Object} summarise()
Produces a summary object of the unit tests performed. This function should only be called when the unit tests have completed. The summary object contains four fields:
  • name - a string set to the unit test name.
  • abend - a string which is set if an abnormal condition (such as unit testing timeout) occurred.
  • summary - an object containing the actual summary information.
  • tests - an object containing individual test function results.
Returns:
{Object} the summary object.
