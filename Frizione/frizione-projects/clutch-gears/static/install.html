Clutch-Gears Install Checks

The 'Install Gears now' button below appears only if Gears is not installed. A message is also displayed to show if Gears is already installed, or not.

Gears is installed.  Gears is not installed.

With Gears installed, click on the ask permission button below. If you've already given permission, the button won't appear, so remove permission for http://clutch.syger.it (or whatever URL you've mapped localhost to), then refresh the page.
